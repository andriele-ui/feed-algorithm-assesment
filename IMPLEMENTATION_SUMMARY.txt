🚀 PRÓXIMAS ETAPAS (Requer ação no GitHub)

1️⃣  CRIAR REPOSITÓRIO NO GITHUB
    → Acesse: https://github.com/new
    → Nome: "feed-evaluation"
    → Visibilidade: PUBLIC
    → Clique "Create repository"

2️⃣  FAZER PUSH DO REPOSITÓRIO LOCAL
    
    Cole estes comandos no terminal (substitua {seu-usuario}):
    
    cd /Users/andrielelitenski/Development/Feed_evaluation
    git remote add origin https://github.com/{seu-usuario}/feed-evaluation.git
    git branch -M main
    git push -u origin main

3️⃣  ATIVAR GITHUB PAGES
    → Vá para Settings → Pages do repositório
    → Source: Branch "main" + Folder "/"
    → Save
    → Aguarde 1-2 minutos para geração da URL

4️⃣  VALIDAR FUNCIONAMENTO
    → Abra https://{seu-usuario}.github.io/feed-evaluation/
    → Cole um token de user_links.csv
    → Verifique redirecionamento e links de versões

5️⃣  COMPARTILHAR LINKS
    → Abra user_links.csv
    → Substitua "GITHUB_USERNAME" pelo seu login GitHub
    → Envie links personalizados aos 48 usuários
